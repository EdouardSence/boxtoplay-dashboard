# Getting Started — boxtoplay-dashboard

## Prérequis

- Node.js (version compatible dépendances modernes, voir `package.json`)
- npm

## Installation

```bash
npm install
```

## Lancer en dev

```bash
npm run dev
```

## Build / test

```bash
npm run build
npm run test
```
